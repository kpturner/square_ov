# Official Visits

## What this feature does

As a Director of Ceremonies you will be responsible for managing official visits. This feature allows you to create and manage official visits, including the officers and order of the procession. In addition the following outputs are available:

- Seating reservations for the Lodge Room
- An expected attendance list for the visit

all these things come in a print friendly format so you can work from paper or from your device.

## How do I create an official visit?

The home page will contain your list of official visits. These can be completely fictitious if you are playing or experimenting, or they can be real official visits that you are managing. Only you can see these official visits, unless you choose to share them with other officers. This means that you can play around without impacting anyone else.

Official visits can be for the Craft or for the Royal Arch. You can toggle between the two orders using the provided radio buttons. The system will remember your preference for the next time you visit the page.

Click the "Add Official Visit" button to create a new official visit. You will have two options, either to select an official visit from the master list of official visits for the year in question, or to create a new official visit from scratch.

In the first instance you will be prompted for the masonic year that you are interested in. By default it will be the the current masonic year, which changes after each AGM. You can also select a previous year if you are interested in creating an official visit from a previous year.

Below that is a searchable dropdown box for you to find and select the OV that you want to manage. It can be one for which you are the allocated DC, or it can be any other official visit that you are interested in. Once you have selected the official visit, click the "Save" button to create it. When an official visit is created in this way, the allocated officers and VIP will be automatically added to your official visit, and you can then make any changes that you require.

Alternatively you can just enter a name and date for an official visit that you want to create from scratch. Once entered, click the "Save" button to create it.

## What are the options I can take for my official visit?

The home page will list your official visits and the date that the will take (or took) place. You have the following options that are provided in the form on action buttons next to each official visit:

- Edit - From here you will be able to change the name and date of the official visit.
- Copy - Create a copy of your official visit
- Export - This allows you to copy your OV onto another users homepage for collaboration etc
- Procession - This is the main part of the system, where you can manage the officers and order of the procession.
- Reservations - Here you can print out the seating reservation cards (and spares) for the officiers and VIPs attending the official visit.
- Attendance - Here you can print out the expected attendance list for the official visit, which is useful for the DC and the Secretary to manage the visit.
- Delete - This will delete the official visit from your homepage. You will be prompted to confirm this action before it is completed.

## When to use it

This is the fundamental reason for using this tool. It is used by Directors of Ceremonies to manage official visits, and in particular the procession. You can create official visits from scratch or by selecting from the master OV list for the year in question. You can also create official visits from the official visit list of a previous year, which is useful if you have a recurring visit.
